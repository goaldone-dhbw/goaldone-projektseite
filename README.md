# Projektseite von Goaldone

erreichbar auf [https://projekt.goaldone.de](https://projekt.goaldone.de)

Man kann die Projektseite mit einem beliebigen Texteditor bearbeiten, da die Dokumente in Markdown geschrieben sind.
Lokale testen kann man MkDocs mit dem Befehl `mkdocs serve` im Terminal nachdem ihr `pip install mkdocs-material` ausgeführt habt.
