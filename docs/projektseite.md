## Tech-Stack:

Für unsere Projektseite verwenden wir MkDocs mit dem Material Theme. 
MkDocs ist ein statischer Site-Generator, der speziell für die Erstellung von Projektseiten und Dokumentationen entwickelt wurde. 
Dokumente werden in Markdown geschrieben aber auch Dateien können veröffentlicht werden.

Gehostet wird die Seite auf Netlify und die Inhalte werden über GitHub verwaltet. 
Wenn eine Änderung auf den master branch gepusht wird, wird die Seite automatisch neu gebaut und aktualisiert.
